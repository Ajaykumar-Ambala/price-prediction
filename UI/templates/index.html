About the ML Model

The model predicts house prices using **Linear Regression** trained on Bengaluru property dataset.

  • 📌 Dataset: Bengaluru House Price Dataset
  • 📌 Algorithm: Linear Regression
  • 📌 Techniques: One-Hot Encoding, Feature Engineering
  • 📌 Cleaned outliers using sqft-per-BHK filtering
  • 📌 Model deployed with Flask API
  • 📌 Avg accuracy: 85–90%

Optimized for fast, real-time predictions.
